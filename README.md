## 她能做什么

**带人设干真活。** 读写文件、全文检索、执行长任务、联网抓取——产线级 Agent 能力开箱即有；套上人格外壳后，这一切由你的角色以她的口癖完成。工作产物保持专业格式，角色感只体现在她对你说的话里。

**分镜先行。** 复杂任务先进入「分镜」模式：她只读不写，画出完整的执行方案给你过目；你点「开始清稿」，她才切回「执笔」动手。中途还能派出「分身」并行干活、「探子」先去把资料翻一遍再汇报。

**印章即权限。** 每次要动你的文件、跑命令、访问网络之外的资源，她都会举着朱红印章等你表态：**准**（这一次）、**全权委托**（这一类以后都行）、**不行**。权限规则可在设置里逐条精调，默认宁严勿松。

**长对话不失忆。** 上下文逼近上限时自动整理成一张「前情提要」纸片；改错了点「回到这一格」，对话与文件改动一起回滚；长活的进度挂在「桌角便签」上逐条打勾。

**角色卡与长期记忆。** 兼容旧版 `.2cy.json` 角色卡；她跨会话记得你的喜好与习惯，记忆明文存本地、可查可删可关。

**一切在本地。** BYOK 自带模型钥匙（支持数十家提供商），数据不经过任何中间服务器，卸载 = 删除文件夹。

## 安装

前往 [Releases](../../releases) 下载对应平台的 zip：

| 平台 | 文件 | 启动 |
|---|---|---|
| Windows | `2cy-agent-windows-x64-*.zip` | 解压后双击 `启动-2CY.bat`。SmartScreen 提示时点「更多信息 → 仍要运行」（未购签名证书所致，非病毒） |
| macOS (Apple Silicon) | `2cy-agent-darwin-arm64-*.zip` | 首次「右键 `启动-2CY.command` → 打开」，之后可直接双击（未做公证所致） |
| Linux | `2cy-agent-linux-x64-*.zip` | `./启动-2CY.sh` |

无需安装 Node、Bun 或任何运行时——单个可执行文件，浏览器自动打开界面。

## 快速开始

1. 启动后在界面中填入你的模型 API Key（BYOK，支持 OpenAI / Anthropic / DeepSeek / Kimi / 智谱 / 通义等数十家）；
2. 直接开聊，或丢给她一个任务：「把这个文件夹里的资料整理成一份周报」；
3. 复杂任务建议先切「分镜」模式，看过方案再放行。

## 角色卡

两种方式给她装上人设：

**放文件**：把角色卡（旧产品导出的 `角色名.2cy.json` 即可）改名为 `character.2cy.json`，放到 `~/.local/share/2cy/`（Windows 为 `C:\Users\你\.local\share\2cy\`），新开会话生效。换角色换文件，摘人设删文件。

**直接说**：「把你的口癖改成××」「你现在是《××》里的××」——她会自己更新角色卡（经你盖章）。

分身与探子不带人设，这是刻意设计：省 token，也不污染检索与摘要结果。

## 长期记忆

她在对话中值得记住的事会经 `remember` 工具沉淀到本地 `~/.local/share/2cy/memory.json`，明文可查。对话里随口维护：「你都记住我什么了」「忘掉第 3 条」「把对我的印象改成××」。设置中可整体关闭。

## 技能

在 `data/skills/技能名/` 放一份 `SKILL.md`（可附脚本与素材），她按需加载——几十个技能也只占几行常驻上下文。加载前同样要过你的印章。你甚至可以让她给自己写技能。

## 常见问题

**为什么杀毒软件/系统拦截？** 程序未购买代码签名证书与苹果公证（个人项目成本考量），按安装表格中的方式放行即可；介意者可从源码自行构建。

**数据会上传吗？** 不会。除了你配置的模型 API 之外，程序不与任何服务器通信。

**手机能用吗？** 当前版本不提供局域网接入，后续视需求评估。
