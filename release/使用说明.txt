2CY Agent 使用说明
====================================

■ Windows
  双击「启动-2CY.bat」。
  如果出现蓝色 SmartScreen 提示：点「更多信息」→「仍要运行」。
  （本程序未购买微软代码签名证书，因此系统会提示。并非病毒。）

■ macOS  ★ 第一次启动请务必按这个顺序 ★
  1) 【右键】点击「启动-2CY.command」→ 选择「打开」（不要双击！）
  2) 弹出的对话框里点「打开」
  3) 如果提示「已损坏」或直接闪退，请打开「终端」App，输入下面这行
     （xattr 后面有空格，然后把整个 2CY 文件夹拖进终端窗口，再回车）：

        xattr -cr 

     完成后再回到第 1 步。

  之后每次启动直接双击即可。

  为什么这么麻烦？本程序没有购买 Apple 公证（每年 99 美元），macOS 会对
  未公证的程序设卡。上面的操作只是告诉系统「这个我信任」，程序本身完全在
  你的电脑本地运行。

■ Linux
  终端里运行：./启动-2CY.sh

====================================

启动后浏览器会自动打开界面。关闭那个黑色窗口即可停止程序。

■ 第一次使用：填入模型 API Key
  界面里进入设置，填入你自己的模型 API Key（支持 OpenAI、Anthropic、
  DeepSeek、Kimi、智谱、通义等数十家）。

■ 让她带上人设（角色卡）
  把角色卡文件改名为 character.2cy.json，放到：
    Windows：C:\Users\你的用户名\.local\share\2cy\
    macOS/Linux：~/.local/share/2cy/
  （文件夹不存在就新建一个。）新开会话即生效。
  也可以直接在对话里说「把你的口癖改成××」，她会自己改卡。

■ 她的记忆
  存在同目录的 memory.json，明文可查可删。
  对话里说「你都记住我什么了」「忘掉第 3 条」也有效。

■ 数据安全
  本程序本地优先，除了你配置的模型 API，不与任何服务器通信。
  卸载 = 删除整个文件夹。
