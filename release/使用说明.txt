2CY Agent 使用说明
==================

【启动】
Windows：双击「启动-2CY.bat」。若出现蓝色 SmartScreen 提示，点「更多信息」→「仍要运行」（程序未购买微软签名证书所致，并非病毒）。
macOS：第一次请「右键点击 启动-2CY.command → 打开」，之后可直接双击。（程序未做苹果公证所致。）
Linux：终端运行 ./启动-2CY.sh

启动后浏览器会自动打开操作界面。关闭方法：关掉启动它的那个黑色窗口即可。

【接入模型（BYOK）】
首次使用需在界面里配置你自己的模型 API Key，支持数十家提供商。

【角色卡】
把角色卡文件改名为 character.2cy.json 放到：
  Windows: C:\Users\你的用户名\.local\share\2cy\
  macOS/Linux: ~/.local/share/2cy/
（文件夹不存在就新建。）新开会话生效。也可以直接在对话里说「把你的口癖改成××」让她自己改卡。

【记忆】
她记住的内容存在同目录 memory.json，明文可查可删；对话里说「你都记住我什么了」「忘掉第 3 条」同样有效。

【数据都在本地】
本程序本地优先，不经过任何中间服务器；卸载 = 删除整个文件夹。
